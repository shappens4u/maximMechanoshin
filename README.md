# Проект "Карта Детства" Находка
style.css - содержит стили сайта и его составляющих
index.html - тело сайта (внешняя оболочка, скелет сайта)
script.js - внутренности тела (содержит уже техническую рабочую часть)
package.json - он компилирует файлы, чтобы вся структура из файлов работала

### Как запустить:
1. `npm install`
2. `npm start`
